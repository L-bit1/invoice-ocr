OCR 证件识别工具 - Windows 版使用说明
========================================

📦 本包为精简版，仅包含配置文件和说明。
完整版（含 OCR.exe）请从 GitHub Releases 下载。

1. 获取完整版 OCR.exe
   
   方法一：从 GitHub Releases 下载（推荐）
   · 访问项目 GitHub 仓库的 Releases 页面
   · 下载「OCR_Windows.zip」（包含 OCR.exe + 配置文件）
   · 解压后即可使用，无需安装 Python
   
   方法二：使用 GitHub Actions 自动构建
   · 访问项目 GitHub 仓库 → Actions 标签
   · 选择「构建 OCR 证件识别工具」工作流
   · 点击「Run workflow」→「Run workflow」
   · 等待构建完成（约 5-10 分钟）
   · 在 Artifacts 中下载「OCR_Windows_完整版.zip」
   
   方法三：本地构建（需 Windows + Python）
   · 在 Windows 电脑上打开「发票软件」项目
   · 进入「OCR单一功能版」目录
   · 双击运行 build_ocr_exe.bat
   · 将生成的 OCR.exe 复制到与本说明同一目录

2. 配置百度 OCR
   将 baidu_ocr_config.example.json 重命名为 baidu_ocr_config.json，
   用记事本打开，填入你在百度智能云申请的 AppID、API Key、Secret Key。

3. 使用
   双击 OCR.exe 启动，在软件内可进行证件识别与税局导入。

解压后运行即可，无需安装 Python。
